为什么需要 Windows 启动管理器呢？

Windows 引导管理器有时也称为引导加载程序。原因是技术性的。您的所有文件（包括操作系统和其他关键系统数据）在断电时都存储在硬盘上。

现在，当您打开系统时，引导管理器的工作就是将您的操作系统（连同其他文件）带到计算机的 RAM 中，以便它可以启动。事实上，除了操作系统本身，启动管理器还可以让您将设备启动到 CD/DVD 驱动器、USB 驱动器或软盘驱动器。

或者正如微软在他们的博客上所说的那样：

    “Windows 启动管理器是 Microsoft 提供的用于设置启动环境的 UEFI 应用程序。在引导环境中，由引导管理器启动的各个引导应用程序在设备引导之前为所有面向客户的场景提供功能。”

如您所见，启动管理器在 Windows 的工作中起着非常重要的作用。基本上，它是 Windows 引导加载体系结构的一部分，除了 Windows 引导管理器 (Bootmgr.exe) 本身之外，它还包括 Windows 操作系统加载器 (Winload.exe) 和 Windows 恢复加载器 (Winresume.exe)。

引导管理器存储在根目录中，不仅负责启动您的 PC，还执行许多其他操作，例如：

    显示屏显示启动菜单。

    加载系统特定的引导加载程序。

    将引导参数移动到引导加载程序。

你能禁用引导管理器吗？

现在，尽管 Boot Manager 工具很方便，但它并非没有缺点。启动管理器的最大障碍之一是它大大减慢了您的 Windows 启动过程。而且，虽然由于明显的技术原因您无法将其从计算机中删除，但您可以做的是调整其设置，以减少启动 PC 所需的时间。这是如何做：

    启动 Windows 控制面板。转到开始菜单搜索栏，输入“控制面板”，然后选择最佳匹配。

    选择System and Security，向下滚动到管理工具并单击它。

    从那里，打开System Configuration。

    现在转到“系统配置”对话框中的“引导”选项卡。

    选择特定的操作系统并将超时设置设置为尽可能低的数字。

    最后，单击OK或Apply。



一旦你这样做，你的 Windows 启动速度将得到相当大的提升。

选择操作系统

正如我们上面所说，引导管理器可以帮助您在双重引导期间选择您的操作系统（在一台机器上运行多个操作系统）。现在，如果您安装了两个或多个操作系统，那么重要的是您必须以一种允许您在每次启动时选择要运行的操作系统的方式进行设置。您可以这样做：

    转到开始菜单搜索栏，输入“msconfig”，然后选择最佳匹配。

    前往引导选项卡，然后从可用操作系统列表中删除默认操作系统选项。

从现在开始，每当您启动 PC 时，系统都会要求您选择操作系统。

Windows 启动管理器

引导管理器是运行 Windows 操作系统的重要方面。这就是为什么微软设计的东西让你根本无法禁用或删除它。希望这个简短的指南能帮助您清楚什么是引导管理器，以及为什么它不仅是 Windows 的重要组件，也是其他操作系统的重要组件。
